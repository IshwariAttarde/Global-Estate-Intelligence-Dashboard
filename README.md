🌍 Global Estate Intelligence Dashboard

📌 Project Overview

The **Global Estate Intelligence Dashboard** is an interactive Power BI project designed to analyze global real-estate market trends across multiple countries including Australia, United States, United Kingdom, Canada, Germany, India, Singapore, and United Arab Emirates.

This dashboard helps users explore property pricing, rental trends, amenities, furnishing status, energy ratings, and market performance through dynamic and visually interactive reports.

---

🎯 Project Objectives

* Analyze global property listings and market trends
* Compare rental and sales performance across countries
* Understand pricing patterns using Price Per SQM metrics
* Track property amenities and furnishing status
* Build interactive and modern Power BI dashboard experiences
* Create custom HTML/SVG visuals inside Power BI tables

---

📊 Dashboard Pages

This page provides a high-level overview of the real-estate portfolio.

### Key Insights:

* Average Days on Market

This dashboard focuses on pricing trends and market intelligence.

This page provides operational and property-level details.
